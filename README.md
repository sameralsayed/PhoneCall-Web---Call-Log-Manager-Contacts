# 🌟 PhoneCall Web - Call Log Manager & Contacts

**A sleek, fully functional browser-based Call Log Manager & Contacts web app that simulates the popular Android app "Phone Call" (phonecall.log.manager.contacts).**

View complete call history, manage contacts with favorites & search, block unwanted numbers, experience the After-Call Screen, and enjoy one-tap actions — all running 100% locally in your browser.

## ✨ Features
- 📜 **Complete Call History** – Incoming, outgoing, missed with filters  
- 📞 **One-Tap Call Back** – Click any log to simulate calling  
- 👥 **Contacts Manager** – Smart search, add to favorites  
- ⭐ **Favorites Section** – Quick access to important contacts  
- 🚫 **Block List** – Block & unblock numbers instantly  
- 📱 **After-Call Screen Demo** – Full modal with actions (call, message, save, block)  
- 🔍 **Smart Search** – Real-time contact filtering  
- 📊 **Call Details** – Duration, time, type shown clearly  
- 📱 **100% Mobile-First** – Beautiful on phones, tablets & desktop  
- 🌐 **SEO Optimized** – Rich meta tags, keywords & semantic HTML  
- 🎨 **Clean Modern UI** – Blue/green accents, smooth animations
